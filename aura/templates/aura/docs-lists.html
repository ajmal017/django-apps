Docs - Lists

Connect lists-icon-boxes.css to use lists styles.

List appearance could be changed using next modifyers:

  • .pi-no-bullets - no bullets list
  • .pi-list-with-icons - lists with icons
  • .pi-bullets-circle - lists with icons that are inside a circle
  • .pi-bullets-square - lists with icons that are inside a square
  • .pi-bullets-base - lists with icons that are inside a circle or square in base theme color
  • .pi-list-dashed - lists with dashed borders
  • .pi-list-icons-ok - list items with "ok" icon
  • .pi-list-icons-heart - list items with "heart" icon
  • .pi-list-icons-star - list items with "star" icon
  • .pi-list-icons-plus - list items with "plus" icon
  • .pi-list-icons-minus - list items with "minus" icon
  • .pi-list-icons-link - list items with "link" icon
  • .pi-list-icons-right-open - list items with "right-open" icon
  • .pi-list-icons-right-dir - list items with "right-dir" icon
  • .pi-list-icons-news - list items with "news" icon

List items appearance could be changed using next modifyers:

  • .pi-bullet-icon - list item icon
  • .pi-bullet-icon-img - list item image
